# PodPulse
- Listen Different Podcast.
- User can upload their own podcast.
- User can search podcast based on different category.
- User-friendly UI.
  ---
## Live Link : https://podpulse-39t5.vercel.app/

## Tech Stack
- ReactJs
- NodeJs
- Express
- MongoDB
- Redux
